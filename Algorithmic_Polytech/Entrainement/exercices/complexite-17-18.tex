prend en entrée un entier $n$.

    Pour i allant de 1 à n:
        Pour j allant de 1 à i:
            c <- c + 1
            
        Pour j allant de i à n:
            c <- c + 1
            
Algo_b :
    c <- 1
    Tant que c < n:
       c <- c*2   
.       
\end{lstlisting}
\end{minipage}
\begin{minipage}[t]{0.48 \textwidth}

Algo_c :

Algo_d :

    Pour i allant de 1 à n:
